We will try to fit a line to the data points $$(-2,-1), ~ (-1,-1), ~ (0,1), ~ (1,2), ~ (2,2), ~ (3,4), ~ (4,3)$$

We'll represent this data in a data-frame, with columns labeled $Y$ and $Y$.values.

We want to represent the problem as a matrix-vector equation. The R function model.matrix() will do this for us.

We'll also need the transpose of the matrix, $A$. The function t(A) will do this for us.

Now we will solve the linear system $$A^\top Ax = A^\top b.$$ For that we have to use matrix-matrix and matrix-vector multiplication. In R, this is done with the %*% operator. We'll also use the solve() function to solve the linear system.

In [33]:
(u=solve(t(A) %*% A ,t(A)%*%b))
(Intercept)0.8214286
X0.7500000
