Course Content: The course combines a systematic study of operating systems, with key elements of programming (e.g., process and resource management) which support that study. There will be a particular emphasis on C programming, though some Python will also be used.
Assessment: Regular lab assignments and a 2 hour written exam at the end of Semester 2.
Text books: Recommended texts include
  • Operating Systems: Three Easy Pieces, by Remzi H. Arpaci-Dusseau and Andrea C. Arpaci-Dusseau. This text is free; individual chapters can be downloaded in PDF form from http://pages.cs.wisc.edu/~remzi/OSTEP/.
  • Silberschatz et al, Operating System Concepts. There are many copies in the library.
  • King, C Programming - a modern approach (2nd Ed).
