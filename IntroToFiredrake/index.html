Introduction to Firedrake

Notebooks and scripts to help getting started with Firedrake

This is a selection of Jupyter Notebooks and (later) Python scripts that demonstrate some capabilities of Firedrake, the finite element system for solving partial differential equations (PDEs) using finite element methods (FEMs).

These notebooks and scripts were developed by Niall Madden for members of his team. If they are useful to you, great! If you spot any bugs, let Niall know

Philosophy: Jupyter is an amazing system! And the Introductory tutorials are a great place to start, especially if you are already very familiar with PDEs, and with FEMs. The goal of these notebooks is to be even more introductory, so that that are accessible to (say) undergraduates who are just learning about differential equations, and about FEMs. Knowledge of Python is assumed.

Getting started: The easiest way to start programming with Firedrake is to use Google collab. Follow the instructions at FEM on Colab
